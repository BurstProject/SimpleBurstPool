Simple Burst Pool


Register payout address:


Network State
Current share counts


Specs:
Proportional payouts(pool hopping really isn't an issue here)
2% fee
Payout occurs instantly after a block is found

Usage:
Paste your BURST- address into the box at the top to receive a generate address.
Generate plots for that address, and use pocminer_pool to mine with them
windows: java -cp pocminer_pool.jar;lib/*;lib/akka/*;lib/jetty/* pocminer_pool.POCMiner mine http://127.0.0.1:8125 http://thispoolsip
linux: java -cp pocminer_pool.jar:lib/*:lib/akka/*:lib/jetty/* pocminer_pool.POCMiner mine http://127.0.0.1:8125 http://thispoolsip
Local wallet does not have to be running, but if it is pocminer_pool will solo mine with plots it has passphrases for

Temp bans:
Due to verification of shares being computationally expensive, the pool will temp ban ips that submit bad results multiple times
Bans only last 10 minutes
Multiple occurances of incorrect work, or duplicate shares in a short period of time will trigger temp bans
